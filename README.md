# orderbook-reconstruction

A C++ project which implements an orderbook and outputs the best bids and asks.



## Run on Terminal

### Orderbook 1
```sh
g++ orderbook1.cpp -o o1
o1
```

### Orderbook 2
```sh
g++ orderbook2.cpp -o o2
o2
```
